# Scanner de IP - TunnelRat Oficial

Este é um projeto simples em Go para capturar informações de rede, desenvolvido como parte dos meus estudos e Segurança Cibernética.

## 🚀 Como usar
Basta baixar conforme seu sistema operacional e rodar na mesma rede . Caso queira enviar para rede diferente use o programa cloudflared para criar o tunelamento na porta 4444.

1: Na sua rede local basta rodar o programa no seu navegador ou celular colocar
http://127.0.0:4444 ou http://localhost:4444.

2: Se for mandar para o golpista fora da sua rede usar o programa Cloudflared como
tunelamento, ou abrir a porta 4444 do seu roteador porem tome cuidado!!!
